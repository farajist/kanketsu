Minimal REST API project starter using express and typescript

## Installation and Setup Instructions

Clone down this repository. You will need `node` and `yarn` installed globally on your machine.

Installation:

`yarn`

To Run Lint (eslint):

`yarn lint`

To Run Test Suite:

`yarn test`

To Start Server:

`yarn start`
